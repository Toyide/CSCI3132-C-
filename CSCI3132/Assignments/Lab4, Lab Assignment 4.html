1. Lab exercise

In Java or C# this is called generic. However, template in c++ is quite different than those, they are infinitely more powerfully.

The compiler writes code for you. Let us see an example of writing three functions for printing an integer number, a float number, and a string into the console.

#include <iostream>
#include <string>

#define LOG(x) std::cout << x << std::endl;

// 1. Declare and impelement a print function that prints an integer number.
// 2. Declare and impelement a print function that prints a string.
// 3. Declare and impelement a print function that prints a float number.

int main()
{
    Print(5);
    Print("Reza");
    Print(5.5f);
    return 0;
}

Let us use a template function to make it much easier.

#include <iostream>
#include <string>

#define LOG(x) std::cout << x << std::endl;

// 1. Try to use templates to have only one function that prints all three

int main()
{
    Print(5);
    Print("Reza");
    Print(5.5f);
    return 0;
}

2. Lab assignment 4

The aim is to develop a template vector class that can handle any data type. Create a template class vector using the header file, "vector.h" and submit the "vector.h" file. Please note that the main file is provided in the assignment folder. Also, note that both the declaration and the implementation needs to be done in the "vector.h" file. Please make sure you have completed the followings tasks:

Marking Scheme:
1- Testing the Default constructor that creates a vector of size zero of the types either integer numbers, float numbers, strings and etc. (5%)
2- Testing the attribute data in the vector class (5%)
3- Testing the overloaded constructor which inputs the size (5%)
4- Testing the overloaded constructor which inputs an array (5%)
5- Testing the overloaded array operator "[]" (5%)
6- Testing the overloaded shift operator "<<" (5%)
7- Testing the assign function (5%)
8- Testing the add function (5%)
9- Testing the overloaded add operator "+" (5%)
10- Testing the resize function (5%)
--- Clean code and commenting (10%)
--- Declare and implement the following functions in your vector class:
11- BubbleSort: (10%)
This function sorts the data array using bubble sort algorithm.
12- Rotate: (30%)
This function should perform a left rotation on a range of elements in the data array. The function takes three pointers as input (first, n_first, last). and
-- first - pointer to the beginning of the original range.
-- n_first - pointer to the element that should appear at the beginning of the rotated range.
-- first - pointer to the end of the original range.
this function swaps the elements in the range [first, last) in such a way that the element n_first becomes the first element of the new range and n_first - 1 becomes the last element.
Hint 1: Use std::iter_swap to swap elements of the data array
Hint 2: Use recursion in your function implementations. Therefore, this function should return a pointer to the n_first++.

important notes

  • Please do not change the main file
  • Only submit the "vector.h" file. (for template functions you should both declare and implement your class in the header file)
  • Do not include the main function in your "vector.h" file.
  • If your code did not compile for any of the tasks you will get a zero for that task.
  • If you had a memory leak in any task you will lose one point in that task
  • Please note that the sort and the rotate algorithms available in the standard template library are not allowed in this assignment.
  • Rotate and Sort function should perform an in-place operation.
